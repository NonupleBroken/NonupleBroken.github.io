p q 选取不当

适用情况：

当 ppp 和 qqq 选取不当时，例如 qqq 选取了 ppp 的 ttt 倍的后几个素数，甚至 qqq 就是 ppp 的后几个素数，可以在较短时间内分解 nnn。

原理

不妨设：

q=tp+kq = tp + k q=tp+k

那么：

n=pq=p(tp+k)=tp2+pkn = pq = p(tp + k) = tp^2 + pk n=pq=p(tp+k)=tp2+pk

即：

tp2+kp−n=0tp^2 + kp - n = 0 tp2+kp−n=0

则：

p=−k+k2+4tn2tp = \frac{-k + \sqrt{k^2 + 4tn}}{2t} p=2t−k+k2+4tn​​

由于 ppp 是整数，所以 k2+4tn\sqrt{k^2 + 4tn}k2+4tn​ 是整数，即 k2+4tnk^2 + 4tnk2+4tn 开方后是整数。

寻找到符合条件的 kkk，那么：

p=−k+k2+4tn2tq=tp+k\begin{aligned} p & = \frac{-k + \sqrt{k^2 + 4tn}}{2t} \\ q & = tp + k \end{aligned} pq​=2t−k+k2+4tn​​=tp+k​

如果 pq==npq == npq==n，那么已成功分解。否则继续寻找 kkk。

代码

大数精确开方需要用到 gmpy2 库中的 iroot。

多因子

适用情况：
nnn 是由两个以上的素数相乘得到，会有些特殊。

原理

如果选取两个以上的素数，记为 p1,p2,p3...p_1, p_2, p_3...p1​,p2​,p3​...，它们相乘得到 nnn，那么：

φ(n)=(p1−1)(p2−1)(p3−1)...\varphi (n) = (p_1 - 1)(p_2 - 1)(p_3 - 1)... φ(n)=(p1​−1)(p2​−1)(p3​−1)...

公钥、私钥、加解密都与一般 RSA 相同。

选取多因子，虽然会减少生成密钥的时间，但同样也更容易被破解。

共模攻击

适用情况：

当两个用户使用相同的模数、不同的私钥，加密同一明文消息时，即存在共模攻击。

原理

两用户私钥不同而模数相同，则两用户公钥必定不同。

设两用户的公钥分别为 e1e_1e1​ 和 e2e_2e2​，由于选取的 eee 与 φ(n)\varphi (n)φ(n) 互质，因此 e1e_1e1​ 和 e2e_2e2​ 互质。明文为 mmm，密文分别为：

c1=me1 mod nc2=me2 mod nc_1 = m^{e_1} \bmod n \\ c_2 = m^{e_2} \bmod n \\ c1​=me1​modnc2​=me2​modn

当攻击者截获 c1c_1c1​ 和 c2c_2c2​ 后，就可以恢复出明文。

用扩展欧几里得算法可求出 re1+se2=1 mod nre_1 + se_2 = 1 \bmod nre1​+se2​=1modn 的两个整数 rrr 和 sss，由此可得：

c1rc2s=mre1mse2 mod n=m(re1+se2) mod n=(m mod n)(re1+se2) mod n=(m mod n)1 mod n mod n=m mod n\begin{aligned} c_{1}^{r}c_{2}^{s} &= m^{re_1}m^{se_2}\bmod n \\ &= m^{(re_1+se_2)} \bmod n \\ &= (m\bmod n)^{(re_1+se_2)} \bmod n\\ &= (m\bmod n)^{1 \bmod n} \bmod n\\ &= m \bmod n \end{aligned} c1r​c2s​​=mre1​mse2​modn=m(re1​+se2​)modn=(mmodn)(re1​+se2​)modn=(mmodn)1modnmodn=mmodn​

p-1 光滑

原理

光滑数（Smooth Number）指可以分解为小素数乘积的正整数。

当 ppp 是 NNN 的因数，并且 p−1p-1p−1 是光滑数，可能可以使用 Pollard’s p − 1 算法来分解 NNN。

首先根据费马小定理：

如果 ppp 是一个质数，而整数 aaa 不是 ppp 的倍数，则有 ap−1≡1 mod pa^{p-1} \equiv 1 \bmod pap−1≡1modp

则：

at(p−1)≡1t≡1 mod pa^{t(p-1)} \equiv 1^t \equiv 1 \bmod p at(p−1)≡1t≡1modp

可改为等式：

at(p−1)−1=k∗pa^{t(p-1)} - 1 = k * p at(p−1)−1=k∗p

即 at(p−1)−1a^{t(p-1)} - 1at(p−1)−1 是 ppp 的倍数 。

然后根据 Pollard’s p - 1 算法：

如果 p−1p-1p−1 是一些很小质数的乘积，那么 n!n!n! 就能被 p−1p-1p−1 整除。即 n!=t(p−1)n! = t(p-1)n!=t(p−1)。

对于每一个 n=2,3,4,...n = 2, 3, 4, ...n=2,3,4,...，任意选择一个底数 aaa（事实上，可以简单地选择为 2），并计算：

gcd(an!−1,N)gcd(a^{n!} - 1, N) gcd(an!−1,N)

如果结果不为 1 或 NNN，那么就已成功分解 NNN。

但当 nnn 较大时，直接计算 n!n!n! 将会很消耗资源。在遍历 nnn 时，可以简化运算。

因为：

an! mod N=(a mod N)n! mod Na^{n!} \bmod N = (a \bmod N) ^ {n!} \bmod N an!modN=(amodN)n!modN

所以：

an! mod N={(a mod N)2 mod Nn = 2(a(n−1)! mod N)n mod Nn >= 3a^{n!} \bmod N = \begin{cases} (a \bmod N) ^ 2 \bmod N & \text{n = 2} \\ (a^{(n-1)!} \bmod N) ^ n \bmod N & \text{n >= 3} \end{cases} an!modN={(amodN)2modN(a(n−1)!modN)nmodN​n = 2n >= 3​

代码

例子

[NCTF2019] childRSA

可以发现 ppp 和 qqq 是由前 10000 个随机的许多小质数乘积加 1 得到， 即 p−1p-1p−1 为许多小质数的乘积。那么可以利用 Pollard’s p − 1 算法来分解 N。
